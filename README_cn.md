# 智能垃圾分类识别系统

## 项目简介

本项目基于 RDK X5 开发板设计并实现了一套智能垃圾分类识别系统，充分利用其高性能端侧 AI 算力，实现垃圾目标的实时检测与智能分类。系统采用摄像头采集垃圾图像，在本地部署轻量化目标检测模型完成实时推理，无需依赖云端即可快速识别垃圾类别。系统可准确识别可回收物、厨余垃圾、有害垃圾和其他垃圾四大类别，并将识别结果、类别标签及置信度实时显示，为垃圾分类提供智能化视觉辅助。

---

## 项目特点

- 基于 RDK X5 开发板
- 轻量化目标检测模型
- 端侧 AI 实时推理
- 四类垃圾智能识别
