# talent
本项目基于 RDK X5 开发板设计并实现了一套智能垃圾分类识别系统，充分利用其高性能端侧 AI 算力，实现垃圾目标的实时检测与智能分类。系统采用摄像头采集垃圾图像，在本地部署轻量化目标检测模型完成实时推理，无需依赖云端即可快速识别垃圾类别。系统可准确识别可回收物、厨余垃圾、有害垃圾和其他垃圾四大类别，并将识别结果、类别标签及置信度实时显示，为垃圾分类提供智能化视觉辅助。  项目围绕边缘计算与人工智能技术展开，完成了模型训练、模型转换、RDK X5 平台部署及端侧推理等关键环节，实现了图像采集、目标检测、结果展示的完整识别流程。相比传统人工分类方式，本系统具有实时性强、识别准确率高、部署灵活、离线运行等优势，能够充分发挥边缘 AI 在智能视觉应用中的性能优势。同时，系统具备良好的扩展能力。
